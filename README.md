LembreMed
Aplicativo simples de lembrete de medicamentos – Projeto acadêmico

O LembreMed é um aplicativo Windows Forms desenvolvido com o objetivo de auxiliar usuários a lembrar seus horários de medicação, contribuindo para uma melhor adesão ao tratamento.
O app foi criado como parte de um projeto acadêmico alinhado ao ODS 3 – Saúde e Bem-Estar (ONU).
